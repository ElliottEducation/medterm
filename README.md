# 🩺 Elliott MedTerm AI (MVP)

An AI-powered interactive medical terminology parser and learning hub, specifically designed for medical students to master Latin and Greek word roots.

## Features
- **Structural Deconstruction**: Automatically breaks down complex medical terms into Prefixes, Word Roots, Combining Vowels, and Suffixes.
- **Rule Explanation**: Explains the Vowel Drop Rule and Keep-the-Vowel Rule.
- **Plural Derivation**: Derives plural forms based on classical Latin/Greek rules.

## How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
